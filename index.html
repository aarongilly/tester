Reminder for myself

I knocked this out one night after my kid went down. It's not as difficult as I feared. Here are the basic steps:

  1. VS Code
    1. npm init
    2. in package.json > scripts, supply a "start" key with value "node index.js"
    3. npm express install
    4. add a couple routes for fun
  2. GitHub
    1. new repository
    2. name it whatever
  3. Back in VS Code
    1. Source Control > Push to... > add new remote
    2. Source Control > Push to... > (newly added remote)
  4. Glitch
    1. New Project > Import from GitHub
